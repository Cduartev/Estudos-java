# Repositório de Estudos – Java Básico

Este repositório foi criado com o objetivo de estudar e praticar os fundamentos da linguagem Java.
Aqui você encontrará exercícios e exemplos focados exclusivamente na base da programação,
sendo ideal para iniciantes ou para quem deseja revisar conceitos essenciais.

## Conteúdo abordado

Atualmente, o repositório contempla apenas os seguintes temas:

- Estrutura Sequencial
  - Entrada e saída de dados
  - Operações matemáticas
  - Uso de variáveis e tipos primitivos

- Estrutura Condicional
  - if / else
  - Operadores relacionais e lógicos

- Estruturas Repetitivas
  - while
  - for
  - do while

## Objetivo

O objetivo principal deste repositório é consolidar a base da lógica de programação em Java,
servindo como apoio para estudos acadêmicos, cursos e prática pessoal.

## Observações

Este repositório tem fins exclusivamente educacionais.
Novos exercícios e conteúdos poderão ser adicionados conforme a evolução dos estudos.
